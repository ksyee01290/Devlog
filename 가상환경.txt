deactivate (가상환경 나가기)

Get-Command python (파워쉘 파이싼 위치파악)
where python (터미널 파이썬 위치파악)

Remove-Item -Recurse -Force venv (파워쉘 가상환경 폴더제거)
rmdir /s /q venv (cmd 가상환경 폴더제거)
rm -rf venv (mac 가상환경 폴더제거)

python -m venv venv (가상환경 셋팅)
python -m venv venv --without-pip (인터넷 연결 문제 있으면 대체 pip 없이 우선설치)
python -m ensurepip (pip가 가상환경에 없을때)

venv\Scripts\activate (가상환경 활성화)

pip install flask (플라스크 설치)
pip install python-dotenv (.env 파일에 있는 환경변수를 py에서 읽어올수있게해주는 라이브러리)
pip install flask-sqlalchemy (sql 대신 py 코드로 db를 조작할수있게 해줌)
