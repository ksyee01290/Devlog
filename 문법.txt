{{ }}변수 출력
{% if %}조건문 {%endif%}(꼭써줘야함)
{% for %}반복문 {%endfor%}(꼭써줘야함)
{# #}주석

{% extends "base.html"%} (base 상속받기)
{% block content %} (자식페이지가 내용을 채움)
{% endblock %} (마지막에 꼭닫아줘야함)

url_for (Flask가 알아서 올바른 경로를 찾아줌)
예시 : {{ url_for('static', filename='style.css') }}

기타
app.run(debug=True) (코드수정하면 서버 자동 재시작, 오류 나면 브라우저에서 오류 내용표시)

bootstrap 사용링크
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">

navbar navbar-dark bg-dark  > 어두운 네비게이션 바
container                   > 가운데 정렬 + 양쪽 여백
mt-4                        > 위쪽 여백 (margin-top)
nav-link text-white         > 흰색 링크
